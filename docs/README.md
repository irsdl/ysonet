## A note on the generated lists

The gadget list, plugin list, and credits shown in these pages are produced by the tool itself and can change between releases. The live tool is always the source of truth:

- `ysonet.exe --fullhelp` - every gadget and plugin with options.
- `ysonet.exe --credit` - full credit and history.

If a page here looks out of date, trust the command output.
